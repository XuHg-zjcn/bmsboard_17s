# 17串锂电池保护板
使用[PB7170](http://www.pengshenchip.com/ProductsStd_253.html)作为AFE

## 目标特征
### 通信交互接口
- CAN通信
- 电子墨水(显示电量等信息)
- RGB LED
- 蜂鸣器
- 蓝牙通信(不得使用专有固件)
- LoRa通信

### 传感器
- 每节电池电压
- 电流
- 电芯温度(NTC + 二极管组成温度传感器2D阵列)
- 内部空气 温度,湿度,气压
- 加速度计
- 应变计(安装在外壳内壁和电芯表面，使用ADS1243作为ADC)
- 水浸传感器（电极式，用于外壳内壁）

### 保护
- 过压欠压
- 过流保护
- 短路保护
- 过冷过热保护

除短路和严重过流保护外，不应直接切断，必须先通信确认，使负载处于安全状态（如数据已经保存，电动车停在安全区域），或超时后直接切断

### 其他特征
- 数据记录
- 防盗
- 异常提示
- 休眠模式

## 架构
主MCU(直接与AFE连接)与通信MCU分离
